# 🕌 Tasbeh Al Zikr Counter

A simple Islamic **Tasbeh Counter Web App** built using **HTML, CSS, and JavaScript**.
This project allows users to count their zikr (Tasbeeh) digitally while using a simple and clean interface.

---

## ✨ Features

* ✔ Digital Tasbeh Counter
* ✔ Click button to increase count
* ✔ Clear button to reset counter
* ✔ Simple Islamic themed design
* ✔ Ad integration using Google Ad Manager

---

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
* Google Ad Manager Ads
